## INSTALLING & RUNNING                                       
#### Requirements                                                                                      
* `nose`   
    
#### Testing from source root. (no install)   
`$ nosetests --nocapture`

#### Installing from CLI.                                       
`$ git clone git@github.com:GRAYgoose124/irk.git irk`  
`$ cd irk`                                           
`$ python setup.py install`                       

#### Running after installation.
`$ irk`   
(`CTRL+D` to safely quit)
                                     
#### Uninstalling                                                                                           
`$ pip uninstall irk`
